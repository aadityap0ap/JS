JavaScript Iteration Methods

This file explains for...in, for...of, and forEach() with clear rules and examples.

1. for...in

for...in is mainly used to iterate over object keys.

Syntax

for (let key in object) {
  // use key
}

Key Points

  • Iterates over keys (property names)
  • Best suited for objects
  • Returns index when used on arrays
‚ö†Ô∏è Avoid using for...in with arrays
Order is not guaranteed and it iterates over indexes, not values.

Example: Object

const student = {
  name: 'Aditya',
  age: 21,
  course: 'BCA'
};

for (let key in student) {
  console.log(key, student[key]);
}
// name Aditya
// age 21
// course BCA

Example: Array (Not Recommended)

const nums = [10, 20, 30];

for (let index in nums) {
  console.log(index, nums[index]);
}
// index -> 0,1,2

2. for...of

for...of is used to iterate over values of iterable objects.

Syntax

for (let value of iterable) {
  // use value
}

Key Points

  • Iterates over values
  • Works with arrays, strings, maps, sets
  • Cleaner and safer for arrays than for...in
‚úÖ Use for...of when you care about values, not indexes.

Example: Array

const months = ['Jan', 'Feb', 'Mar'];

for (let month of months) {
  console.log(month);
}
// Jan
// Feb
// Mar

Example: String

const name = 'JS';

for (let ch of name) {
  console.log(ch);
}
// J
// S

3. forEach()

forEach() is an array method used to run a function on each element.

Syntax

array.forEach((element, index, array) => {
  // logic
});

Key Points

  • Works only on arrays
  • Does not return anything
  • Cannot be chained
  • Used for side effects (logging, updating UI)
üö® forEach always returns undefined
Do NOT use it when you need a new array.

Example

const nums = [1, 2, 3];

nums.forEach((num, index) => {
  console.log(index, num);
});
